Hexo部署和主题配置

前言

Hexo 是一个快速、简洁且高效的博客框架。 Hexo 使用 Markdown（或其他标记语言）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装配置 Hexo

  1. 安装 Hexo 需要先安装下列应用程序

    • Node.js
    • Git
    npm install -g hexo-cli

  2. 创建项目

    执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件

    hexo init <folder>
cd <folder>
npm install

  3. 配置

    网站

    设置 描述
    title 网站标题
    subtitle 网站副标题
    description 网站描述
    keywords 网站的关键词。 支持多个关键词。
    author 您的名字
    language zh-CN
    timezone Asia/Shanghai

更改 Quiet 主题

  1. 下载主题，themes目录下执行

    git clone https://github.com/79e/hexo-theme-quiet.git

  2. 修改Hexo根目录下的 _config.yml 文件启用 Quiet 主题

    theme: hexo-theme-quiet

  3. 首页文章布局 默认 ‘card’ 可选择 ‘block-card’，如果选择 ‘card’ ，建议将每页展示的文章数量设置为 每页9篇

    index_generator:
path: ''
per_page: 9
order_by: -date
// 下面还有个 10 的 修改为 9
per_page: 9
pagination_dir: page

  4. 下面此项设置为跟我一样即可显示文章的代码高亮

    highlight:
enable: false
line_number: false
auto_detect: true
tab_replace: ''
wrap: true
hljs: true
prismjs:
enable: false
preprocess: true
line_number: true
tab_replace: ''
    • 进入根目录下的source文件夹下创建tags文件夹新建index.md文件
    ---
title: tags
date: 2020-09-19 16:19:22
layout: "tags"
---
    • 进入根目录下的source文件夹下创建links文件夹新建index.md文件
    ---
title: 友情链接
date: 2020-09-19
type: "links"
layout: "links"
comments: false
---
    • 进入根目录下的source文件夹下创建categories文件夹新建index.md文件
    ---
title: 文章分类
date: 2020-11-02
type: categories
layout: categories
---

  6. 文章发布需要写标头

    ---
title: Hexo部署和主题配置
excerpt: 记录Hexo博客的部署和主题配置过程
date: 2024-08-11 15:30:37
categories: 经验
tags: 
    - Hexo
    - 部署
    - 博客
aubot: Gevelopi
aubot_link: 'https://github.com/Gevelopi'
cover: 'https://imgos.cn/2024/08/11/66b86d5e53ab0.png'
comments: false
toc: false
---
    设置 解释
    title 文章标题
    excerpt 文章摘要
    date文章 发布日期
    categories 文章分类
    tags 文章标签
    aubot 博客作者
    aubot_link 博客作者链接
    cover 文章封面图片
    comments 是否开启评论
    toc 是否开启目录

  7. 本主题使用 less 对 css 进行处理，hexo-renderer-less 安装后重新编译即可

    npm i hexo-renderer-less

    ejs 作为 模版语言

    npm i hexo-renderer-ejs

Hexo 指令

  1. 初始化

    hexo init <folder>

  2. 生成静态文件

    hexo generate

  3. 启动本地服务

    hexo server

  4. 部署

    hexo deploy

  5. 清除缓存

    hexo clean

  6. 更新

    npm update

一键部署到 Github (Git)

  1. 安装 hexo-deployer-git。

    npm install hexo-deployer-git --save

  2. 修改 _config.yml 文件中的 deploy 配置。

    deploy:
  type: git
  repo: <repository url>
  branch: [branch]
  message: [message]

    其中， 是你的 Git 仓库的 URL，[branch] 是你想要部署的分支，[message] 是提交信息。

  3. 执行 hexo deploy 命令。

    hexo deploy
